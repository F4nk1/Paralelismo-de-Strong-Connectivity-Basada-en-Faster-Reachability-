# Metodología Experimental

1. **Objetivo**: Evaluar la aceleración y eficiencia de SCC paralelo con VGC.
2. **Algoritmos**: Tarjan (Secuencial), BGSS (Paralelo), VGC (Híbrido).
3. **Métricas**: Tiempo de respuesta (ms), Speedup ($T_1/T_p$), Correctitud (SCC Count).
4. **Hardware**: Multinúcleo con soporte OpenMP.
